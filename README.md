# ClawItUp 🤖

Building with **GitAgent** for Lyzr AI's Hiring Process Project

## 📋 Project Overview

This repository is part of **Lyzr AI's innovative hiring process** that uses **GitAgent** — a universal, git-native AI agent framework. Instead of traditional hiring, Lyzr is inviting candidates to build actual products using their open-source technologies.

### What is GitAgent?

**GitAgent** is a revolutionary framework that treats Git repositories as AI agent containers. Your agent's identity, memory, rules, tools, and skills are all version-controlled files within the repo.

**Key Resources:**
- 🔗 [GitAgent GitHub](https://github.com/open-gitagent/gitagent)
- 📖 [GitAgent Protocol (GAP) Standard](https://www.gitagent.sh/)
- 📚 [Research Paper - GitAgent: Facilitating Autonomous Agents with GitHub](https://arxiv.org/html/2312.17294v1)

### Why GitAgent?

- **Portable:** Ship complete AI agents across teams and platforms
- **Version-Controlled:** Full agent history, rollback, and transparency
- **Framework-Agnostic:** Works with LangChain, AutoGen, Claude Code, and more
- **Unified Standard:** GitAgentProtocol (GAP) brings standardization to AI agents

## 🚀 Project Requirements

### Core Requirement: Use GitAgent

This project **MUST** utilize GitAgent for:
- Agent initialization and lifecycle management
- Skill and tool versioning
- Persistent agent memory and state
- Integration with existing Lyzr products
