# ClawItUp v1.0.0

## Highlights

- End-to-end audit pipeline with live phase streaming in CLI.
- SDK-driven Gitclaw stage execution with captured assistant/tool traces.
- Provider-agnostic model configuration through `agent.yaml` (`provider:model-id`).
- Structured run artifacts for each audit execution under `runs/<run-id>/`.

## Scope

`v1.0.0` ships a report-first audit gate and intentionally excludes automation-heavy integrations (auto patch/issue/PR, dashboard UI, external chat integrations, unbounded whole-repo mode).
